## 基础块

### 基础用法

:::demo
```html
<template>
  <base-title>
    我是内容
  </base-title>
</template>
```
:::

### 小标题

:::demo
```html
<base-title
  type="small"
>
  我是内容
</base-title>
```
:::

### 亮标题

:::demo
```html
<base-title
  type="light"
>
  我是内容
</base-title>
```
:::

### 自定义宽度

:::demo
```html
<base-title
  :width="300"
>
  我是内容
</base-title>
<base-title
  width="300px"
>
  我是内容
</base-title>
<base-title
  width="100%"
>
  我是内容
</base-title>
```
:::
